    Fonctionnement de l'espace ONISEP-mining

    Cet espace permet d'interroger les ressources opendata mises à disposition par l'ONISEP sur le site data.gouv. Pour l'utiliser il est nécessaire de s'authentifier, pour cela il faut être inscrit et validé pour avoir les droits d'accès. La validation est effectué manuellement par mail pendant la phase de test qui est en cours.

    Pour démarrer la recherche, une fois authentifié, cliquez soit sur "Exploration" soit sur l'un des menus déroulants qui apparaitront dans le bandeau ci-dessus (sous le titre du site).

    Trois modes d'accès aux données sont possibles

    • Recherche guidée à partir de modèles d'exploration préexistants
      • Métiers
      • Secteurs d'activité
      • Centres d'intérêt
    • Recherche directe sur les données
      • Formations
      • Vidéos
    • Informations utiles sur les ressources
      • Métiers
      • Formations
      • Centres d'intérêt

    Règles d'accès

    Pour accéder à la ressource il est nécessaire de s'authentifier avec une adresse mail valide, le pseudo fourni n'est utile que pour des fonctionnalités internes.

    Origine des données mises à disposition

    L'ensemble des données concernant les métiers, les formations et les établissements a été fusionné sous la forme d'un système de données liées (LOD) en RDF et inséré dans une base de données locale. Le site permet d'effectuer des requêtes via une interface simplifiée qui collecte les paramètres nécessaires à la construction de requêtes en langage SPARQL. Les données sont compilées à partir du site ["opendata de l'ONISEP"](http://opendata.onisep.fr) et contiennent les objets suivants :

    • formation initiale lycée
    • formation initiale sup
    • structures d'enseignement secondaire
    • structures d'enseignement supérieur
    • fiches de formations
    • fiches métier
